Yet another implementation of Inuzaru tanka generator. See repository.

短歌自動生成装置「犬猿」（星野しずる）のクローンです。

CSV形式で公開したGoogleスプレッドシートのURLを指定することで使用する語彙を差し替えることができます。スプレッドシートにはword列（語彙が含まれる列）、type列（名詞・修飾部・述部のいずれかを含む列）、mora列（語彙の音数を含む列）がある必要があります。有効なURLを入力するたびに語彙が追加されます。

デフォルトで登録されている語彙は本家の犬猿に収録されている語彙を借りたものです。
